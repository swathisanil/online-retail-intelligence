# Dataset Information

**Download:** [UCI Online Retail Dataset](https://archive.ics.uci.edu/dataset/352/online+retail)

Dataset Name:
Copy of Online Retail.xlsx

Dataset Size:
77.2 MB

Description:
This dataset contains transactional retail sales data used for exploratory data analysis (EDA), SQL analysis, customer segmentation, and Power BI dashboard development.

Analysis Performed:
- Data cleaning
- Revenue analysis
- RFM customer segmentation
- Product performance analysis
- Country-wise sales analysis
- KPI dashboarding

Tools Used:
- Python
- SQL
- Power BI

Note:
The raw dataset is not uploaded to GitHub because the file is very large and repositories should stay lightweight.
